Nota: Aún nos encontramos buscando una mejor forma de manejar las dependencias internas de la librería. Hemos tratado de minimizar el uso de librerías externas, las librerías usadas actualmente son las siguientes:
  • primeng y primeicons: PrimeNG es una librería de componentes usada actualmente para los componentes de tabla, dropdown y calendar. Para poder mostrar los íconos necesitados además por estos componentes debe instalarse también Primeicons.
  • fontawesome: Esta es una librería que nos ofrece una amplia colección de íconos. Actualmente se está usando la versión gratuita de la librería.
  • bootstrap: Nos ofrece un gran conjunto de utilidades y estilos CSS. Ojo: Estamos haciendo uso únicamente de las hojas de estilo de Bootstrap, no debe incluirse en el proyecto JQuery para hacer uso de las funcionalidades extras de Bootstrap.
  • file-saver y xlsx: Estas librerías son utilizadas por el ExcelExporterService para descargar archivos de Excel.
