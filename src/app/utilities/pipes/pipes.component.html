Pipes

Los pipes comunes de la librería ERPSAASCMP están expuestos en el módulo PipesModule, por lo que para hacer uso de los pipes deberá primero importarse el módulo de la siguiente forma:

BooleanPipe

Este pipe fue creado para transformar valores booleanos en valores de texto apropiados. Los valores de texto por defecto son 'Si' para true y 'No' para false. No obstante, estos valores pueden ser configurados para mostrar otro texto.

En la siguiente tabla, se demuestra el uso del pipe BooleanPipe. Debe notarse que los valores configurados para verdadero y falso son opcionales, en caso de que cualquiera de los dos sea nulo o indefinido se utilizará el valor por defecto.

Valores por defecto Valores configurados
true {{ true | boolean }} {{ true | boolean : 'Verdadero' : 'Falso' }}
false {{ false | boolean }} {{ false | boolean : 'Verdadero' : 'Falso' }}
Código {{ valorBooleano | boolean }} {{ valorBooleano | boolean : 'Verdadero' : 'Falso' }}
