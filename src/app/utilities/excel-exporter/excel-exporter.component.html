ExcelExporterService

Este servicio permite la exportación de datos como archivos de Excel simples.

El servicio ExcelExporterService ofrece el siguiente contrato:

Método Descripción Ejemplo

Se encarga de exportar la colección dada por parámetro a un archivo de Excel. El archivo de Excel generado utilizará las llaves de los objetos de la colección como encabezados, y utilizará los valores en las llaves para poblar las distintas filas del archivo.

El parámetro excelFileName corresponde al nombre del archivo que se asignará al archivo a generado.

El método además cuenta con dos parámetros opcionales. El primero es sheetName, y corresponde al nombre que se le asignará a la hoja de Excel donde estarán los datos. Si no se especifica el valor de sheetName, se pondrá por defecto el valor 'data'. El segundo parámetro opcional es includeTimestamp y su valor por defecto es false. Si este último parámetro se especifica como true, se pondrá en el nombre del archivo generado el timestamp del momento en el que se generó.
