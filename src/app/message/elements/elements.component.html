Elementos del módulo

Para ofrecer la funcionalidad de mensajes de alerta, proveemos al programador de:

  • Un enumerador de tipos de mensajes.
  • Un modelo de mensaje.
  • Un servicio de mensajes.
  • Un componente para mostrar los mensajes.

Enumerador

El enumerador MessageType ofrece 4 tipos de mensajes que son encontrados frecuentemente en las aplicaciones:

  • Success
  • Error
  • Info
  • Warning

Este enumerador es usado en el modelo de mensaje que permite el uso de la funcionalidad.

Modelo

Para modelar los mensajes que pueden ser mostrados en la aplicación, definimos un simple modelo llamado Message, cuyas propiedades son:

  • text: string
  • type: MessageType

Servicio

La librería provee además un servicio inyectable llamado MessageService, que provee el siguiente contrato:

Método Descripción
Se encarga a añadir un mensaje a la lista de mensajes que están siendo mostrados en la aplicación.
Se encarga de remover de la aplicación el mensaje que se encuentre en la posición dada por parámetro. Este método es más que todo para uso del componente MessageOutletComponent y se recomienda no usarlo si es posible.
Se encarga de retornar el Observable de la lista de mensajes de la aplicación. Este método es más que todo para uso del componente MessageOutletComponent y se recomienda no usarlo si es posible.

Componente

El componente MessageOutletComponent es el que se utilizará para mostrar la lista de mensajes que están registrados en la aplicación.
